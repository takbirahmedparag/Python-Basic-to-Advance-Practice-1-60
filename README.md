# Python-Basic-to-Advance-Practice-1-60
In this project, I practiced some fundamental Python concepts such as comments, print statements, user input, and variables. These basic concepts are important for building a strong programming foundation before moving to more advanced Python topics and data analysis.

## Project Objective
The main objective of this project is to understand the basic structure of Python programming and how simple programs interact with users.
This practice helps to:
•	Understand Python comments
•	Use the print() function to display output
•	Take user input using input()
•	Create and use variables
•	Display text and variables together in output

## Concepts Covered
•	Python comments
•	Print statements
•	User input
•	Variable declaration
•	Output formatting using variables

## Process
•	Wrote comments to explain different parts of the code
•	Took user input such as name, age, and country
•	Stored the input values in variables
•	Used the print() function to display the information
•	Declared additional variables and printed them with messages
